# Lisp-training

Lisp training

# Usage

```
$ git clone git://github.com/thinkphp/lisp-training.git
$ cd lisp-training
$ clisp
$ [1]> (load "sum.lsp")
$ ;; Loading file sum.lsp ...
$ ;; Loaded file sum.lsp
$ [2]> (makesum 100)
$ => 5050
$ [3]> (exit)
```
